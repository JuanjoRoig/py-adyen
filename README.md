# py-adyen
Adyen Python implementation. Capable of both single and recurring payments.

## Installation

Install via pip:
```bash
$ pip install https://github.com/svdgraaf/py-adyen/archive/master.zip
```

## Requirements
Hmac, hashlib. If you want to use the API, you need Suds.
